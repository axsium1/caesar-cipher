# 🔐 Caesar Cipher

> Classical encryption tool written in Python — part of my cybersecurity path.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat&logo=python&logoColor=white)
![Cryptography](https://img.shields.io/badge/Topic-Cryptography-red)

## 🚀 Usage

Run the script:

    python caesar_cipher.py

## ⚙️ Features
- Encrypts and decrypts any text
- Supports uppercase and lowercase letters
- Non-letter characters are preserved

## 🧠 How it works
Each letter is shifted by N positions in the alphabet.
A with shift 3 → D. Wraps around using modulo 26.
